# fakebook back

Prérequis : 

Avoir une connexion internet et un git bash avec un ordinateur fonctionnel. 

Clone du projet :

git clone  https://github.com/lucasynov/fakebookback.git

Installation des modules node : 

cd fakebookback
npm install

Lancement du serveur :

node server


Ports utilisés : 

8001 : socket.io;
8000 : REST API
